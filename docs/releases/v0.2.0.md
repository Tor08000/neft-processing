RELEASE NOTES — v0.2.0 (Client Portal MVP Foundation)
# NEFT Platform — Release v0.2.0
Date: 2025-12-03

## Overview
Первый публичный шаг в сторону клиентского кабинета NEFT. 
Добавлен отдельный клиентский фронтенд, маршрутизация, логин, базовые страницы и инфраструктурная интеграция через gateway и docker-compose.
Обновлена документация и диагностика.

## Highlights
- Новый сервис: `client-web` (Vite + React)
- Страницы: Login, Dashboard, Operations, Limits
- Глобальная структура клиентского UI (Layout, Router)
- Gateway routing для static + API трафика
- Префиксы `/client/` и `/client/api/`
- Docker Compose интеграция client-web
- README дополнен инструкциями по запуску client UI
- Диагностика обновлена: проверка структуры client-web

## Infra Changes
- Новый upstream в gateway/nginx.conf
- Добавлен client-web в docker-compose.yml
- Обновлён набор переменных окружения

## Backend/API Changes
- Создано API для клиентского логина
- Добавлены placeholder-эндпоинты для операций и лимитов клиента
- Подготовлена структура для расширения API

## Diagnostics
- inspect_neft_repo теперь проверяет client-web
- README обновлён
