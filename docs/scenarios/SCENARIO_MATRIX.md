## Pre-External Exclusions (not scenario rows)

The matrix above covers mounted internal owners only. The following contours are intentionally excluded from the internal release gate until a later owner/provider phase:

- partner `/contracts`
- partner `/settlements*`
- client logistics provider-backed fuel-consumption write tails
- marketplace recommendations without mounted owner proof
- marketplace SLA/consequence happy-path tails without mounted owner proof
- provider-backed OTP, email/SMS delivery, live EDO transport, bank API, ERP/1C sync, and external fuel/logistics providers

These exclusions must be documented as frozen/optional contours in readiness docs rather than reintroduced as fake-green scenario rows.
