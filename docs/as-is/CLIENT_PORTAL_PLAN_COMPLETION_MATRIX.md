## 4.1.a Client shell segmentation truth

- Client shell segmentation is driven by `GET /api/core/portal/me`.
- `org.org_type` is the contracted input for `INDIVIDUAL` vs `BUSINESS`.
- Legacy `/api/client/*` business routes still exist as compatibility/public families where consumer migration is incomplete; they must not be confused with bootstrap ownership.
