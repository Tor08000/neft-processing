# Admin Portal · Documents

## Routes

- `/admin/documents`
- `/admin/closing-packages` (closing_package)

## Documents

Capabilities:

- view all documents
- filter by client / period / status
- manual generation trigger
- resend
- void with reason

## closing_package

A closing_package is a bundle of:

- invoice
- act
- reconciliation_act

UI should show:

- package status
- document_acknowledgements (who/when)
- readiness to finalize
