# Admin Portal · Documents

## Routes

- `/admin/documents`
- `/admin/closing-packages`

## Documents

Capabilities:

- view all documents
- filter by client / period / status
- manual generation trigger
- resend
- void with reason

## Closing packages

A closing package is a bundle of:

- invoice
- act
- reconciliation act

UI should show:

- package status
- acknowledgements (who/when)
- readiness to finalize
